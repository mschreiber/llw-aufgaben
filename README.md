Übungsbeispiele für den Lehrlings-Leistungs-Wettbewerb 
</p>
<p>
Hier eine Sammlung von Übungsaufgaben für den Lehrlings-Leistungs-Wettbewerb.<br>
Die Übungsaufgaben haben einen ähnlichen Schwierigkeitsgrad, wie die Prüfungsaufgabe.<br>
Die Lösungen sind Beispielimplementierungen in Java und C#. Sie sollen als Anhaltspunkt für mögliche Lösungen dienen und keinesfalls als "DIE" Lösung gesehen werden.<br>
